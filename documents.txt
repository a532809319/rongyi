这儿是一个ECAHybrid的基础工程,运行说明如下 

git clone  http://192.168.2.30/dragon/ECAHybrid.git  

1.进入到根目录ECAHybrid
  cd ECAHybrid

2.安装配置
  npm install 
  npm install -g webpack-dev-server 

3.运行
  npm start
